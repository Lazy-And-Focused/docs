# Пулл реквесты

## Создание

Название пулл реквеста может быть как [название коммитов](./commits.md)

## Описание

Описание пулл реквеста обязано содержать ссылку на решаемое issue, если такое существует

## Слияние

Слить ветки можно только в том случае, если:

- Пулл реквест был заревьювен
- Код пулл реквеста прошёл все проверки
- Код пулл реквеста соответствует соглашению LAF
